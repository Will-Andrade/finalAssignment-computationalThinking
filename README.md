- a)  a quantidade de pessoas casadas; ✅

- b) a quantidade de pessoas solteiras; ✅

- c) a média das idades das pessoas viúvas; ✅

- d) a porcentagem de pessoas desquitadas ou separadas dentre todas as pessoas analisadas. ✅
